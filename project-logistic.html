Концепт сайта для транспортной компании

В рамках курса «Веб-дизайн для начинающих» от Skillbox. Тема и задача выбраны самостоятельно.

Задача

Сайт для не существующей молодой только набирающей обороты транспортной компании.

• Обязательное условие: удобство пользования сайтом, для постоянных клиентов и для новых.

• Возможность наполнения большей информацией без потери удобства пользования.

• Обязательно: калькулятор доставки.

Принято решение сделать его светлым, чистым, с большим количеством воздуха. На главной странице небольшое понятное меню. Так же предварительный калькулятор доставки. На первом экране есть предложение по перевозке. Это актуально для новых клиентов, которые ищут компанию для транспортировки. Для постоянных клиентов так же есть трекер по отслеживанию груза.

С этой же страницы можем перейти в подробный расчет стоимости доставки, который учитывает множество параметров.

Для Ж/Д доставки и Авиа разные пункты меню. Пользователь выбирает необходимый вариант. Внутри раздела можно выбрать юридическое или физическое лицо, в зависимости от выбранного варианта обновляется контент. Для удобства навигации по огромному количеству информации, все содержимое разделено на соответствующие разделы и помещено в «гармошку». Так у пользователя не будет длинного полотна информации, в котором можно потеряться. Переходя к новой информации, прочитанная закрывается. Там же в разделах могут содержаться прикрепленные файлы, такие как формы для заполнения, перечень документов на отправку груза и т.д. Новые разделы появляясь на странице, не нагромождают ее.

В разделе Адреса терминалов имеется карта для ориентирования, адреса и номера телефонов с кнопкой, сразу для звонка.
